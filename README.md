# Maternal Health Risk Prediction Using Machine Learning

## Project Overview
This project predicts maternal health risk levels using clinical indicators such as blood pressure, blood sugar, heart rate, and body temperature.

## Objectives
- Identify high-risk pregnancies
- Support data-driven healthcare decisions
- Demonstrate machine learning and MLOps skills

## Dataset Features
- Age
- SystolicBP
- DiastolicBP
- BloodSugar
- BodyTemp
- HeartRate

## Target Variable
- Low Risk
- Mid Risk
- High Risk

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Joblib
- GitHub

## Machine Learning Workflow
1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Model Saving
